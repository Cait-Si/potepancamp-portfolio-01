<h1>アプリ名</h1>
<h3>新習慣記録</h3>
<br>
<br>
<h1>なぜ作ったのか</h1>
<h3>早起きやランニング等の良い習慣を新しく取り入れて定着させることはとても難しいことです。そのため、難しい習慣化を少しでもサポートできればと思い作成しました。</h3>
<br>
<br>
<h1>デモ</h1>
<h3>まず、新しく習慣化したい事柄を登録します。その後は実施できた時にできたボタンを押し回数を増やしていきます。回数を記録することで達成感を感じ易くします。</h3>
